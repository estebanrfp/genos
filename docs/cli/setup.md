# `genosos setup`

Initialize `~/.genosv1/genosos.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
genosos setup
genosos setup --workspace ~/.genosv1/workspace
```

To run the wizard via setup:

```bash
genosos setup --wizard
```
